# KAIZEN - Aplicación de Aprendizaje de Lenguaje de Señas

Una aplicación web completa e interactiva para aprender lenguaje de señas, inspirada en Duolingo pero especializada en comunicación visual y gestual.

## 🤟 Características Principales

### ✨ Experiencia de Usuario Especializada
- **Preloader animado** con transición suave
- **Logo TERESIAS** con gestos de manos perfectamente integrado
- **Animación letra por letra** de "KAIZEN - Lenguaje de Señas"
- **Interfaz visual optimizada** para aprendizaje gestual
- **Diseño 100% responsivo** para todos los dispositivos

### 🎯 Sistema de Aprendizaje Especializado
- **6 categorías de aprendizaje**:
  - 🔤 **Alfabeto Dactilológico** (A-Z)
  - 🔢 **Números** (0-100)
  - 👨‍👩‍👧‍👦 **Familia** (mamá, papá, hermanos)
  - 🎨 **Colores** (rojo, azul, verde, amarillo)
  - 😊 **Emociones** (feliz, triste, enojado, sorprendido)
  - 🏃 **Acciones** (comer, beber, dormir, caminar)

### 📚 Tipos de Ejercicios Interactivos
- **Identificación de señas**: ¿Qué seña es esta?
- **Asociación palabra-seña**: Encuentra la seña correcta
- **Opción múltiple visual**: Reconocimiento de gestos
- **Deletreo dactilológico**: Practica el alfabeto
- **Conversación básica**: Frases y expresiones

### 🎮 Modos de Práctica Especializados
- **⚡ Práctica Rápida**: Reconocimiento en 5 minutos
- **🔤 Deletreo**: Alfabeto dactilológico completo
- **🔢 Números**: Del 0 al 100
- **💬 Conversación**: Frases y expresiones comunes

### 📖 Diccionario Interactivo de Señas
- **Búsqueda avanzada** por palabra o descripción
- **Filtros por categoría** (alfabeto, números, familia, etc.)
- **Detalles completos** de cada seña:
  - Imagen/video demostrativo
  - Descripción detallada
  - Consejos de ejecución
  - Botón de favoritos
  - Práctica individual

### 📹 Práctica con Cámara Web
- **Reconocimiento en tiempo real** (simulado)
- **Desafíos aleatorios** de señas
- **Captura y evaluación** de gestos
- **Retroalimentación inmediata**
- **Progreso visual** del aprendizaje

### 🏆 Sistema de Gamificación
- **Sistema de vidas** con regeneración automática
- **Moneda de diamantes** para compras
- **Sistema de rachas** diarias
- **Logros especializados**:
  - Primera Seña
  - Maestro del Alfabeto
  - Racha Semanal
  - Hora de Práctica
  - Nivel 5
  - Coleccionista de Diamantes

### 👤 Perfil Personalizado
- **Estadísticas especializadas**:
  - Señas aprendidas
  - Minutos practicados
  - XP total acumulado
  - Racha actual
- **Sistema de logros** visual
- **Foto de perfil** personalizable

## 🔧 Instalación y Uso

### Requisitos del Sistema
- **Navegador moderno** con soporte para WebRTC
- **Cámara web** (opcional, para práctica avanzada)
- **Conexión a internet** (para carga inicial)
- **JavaScript habilitado**

### Instalación Rápida
1. Descarga todos los archivos
2. Abre `index-sign-language.html` en tu navegador
3. Permite acceso a la cámara cuando se solicite
4. ¡Comienza a aprender lenguaje de señas!

### Uso Avanzado
```bash
# Servidor local con Python
python -m http.server 8000

# Servidor local con Node.js
npx http-server

# Accede a http://localhost:8000/index-sign-language.html
```

## 🎨 Personalización

### Agregar Nuevas Señas
```javascript
// En script-sign-language.js
signDatabase.newCategory = [
    {
        id: 'nueva_sena',
        word: 'Nueva Seña',
        category: 'nueva_categoria',
        image: 'url_de_imagen',
        description: 'Descripción de la seña',
        tips: ['Consejo 1', 'Consejo 2']
    }
];
```

### Modificar Categorías
```javascript
// Agregar nueva unidad de aprendizaje
const nuevaUnidad = {
    title: "Nueva Categoría",
    icon: "fas fa-new-icon",
    lessons: 8,
    color: "#COLOR",
    category: "nueva_categoria"
};
```

### Personalizar Reconocimiento
```javascript
// Ajustar precisión del reconocimiento simulado
const isCorrect = Math.random() > 0.2; // 80% precisión
```
